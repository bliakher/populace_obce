## Population Analysis of Czech Municipalities

Projekt vznikl jako semestrální práce v rámci univerzitního kurzu Data Science. Zaměřuje se na analýzu populačního vývoje obcí v České republice (Liberecký kraj) v období 1995–2020.

### Obsah
- čištění a předzpracování dat
- exploratorní analýza časových řad
- klastrování obcí podle podobnosti populačního vývoje
- interpretace výsledných skupin

### Použité metody
- normalizace časových řad
- klastrování pomocí K-Means
- porovnání časových řad pomocí DTW (Dynamic Time Warping)

### Použité knihovny
- pandas
- numpy
- scikit-learn
- tslearn
- matplotlib
- seaborn

### Cíl
Identifikovat skupiny obcí s podobným demografickým vývojem a popsat jejich charakteristiky.
